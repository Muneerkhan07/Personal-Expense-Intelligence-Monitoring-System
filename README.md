# Personal Expense Intelligence & Monitoring System

## Overview
 
The Personal Expense Intelligence & Monitoring System is a data analytics project designed to monitor, analyze, and visualize personal spending behavior using SQL Server, Python, and Power BI.

The system helps identify high-expense categories, track monthly spending trends, and generate actionable financial insights to support better financial decision-making.

---

## Tech Stack

* SQL Server
* Python (Pandas, NumPy)
* Power BI

---

## Key Features

* Monthly expense trend analysis
* Category-wise spending breakdown
* Interactive Power BI dashboard
* Expense monitoring and tracking
* Smart financial insight generation
* Spending pattern analysis

---

## Business Insights Generated

* Identified top spending categories contributing to overall expenses
* Tracked monthly spending growth trends
* Highlighted overspending behavior patterns
* Generated recommendations for potential savings opportunities

---

## Project Workflow

CSV Data → SQL Server → Python Data Analysis → Power BI Visualization Dashboard
